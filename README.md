# ID-face
<p>淘然视界老年人脸数据（如有问题，可以随时在Issues中提问）：</p>
<p>1）数据规模：当前1000，持续更新</p>
<p>2）数据种类：图片+框标注</p>
<p>3）采集设备：手机、IPC等</p>
<p>4）种族分布：黄种</p>
<p>5）采集场景：室内、室外</p>
<p>6）年龄分布：50+</p>
<p>7）角度姿态：多样</p>
<p>8）人脸表情：多样</p>
<p>9）光照分布：多样</p>
<p>10）数据格式：jpg</p>
